# Garbage Classification

A simple image classification project to classify garbage into different categories. It uses a Convolutional Neural Network (CNN) to classify images of garbage. The project includes a web application to test the model.

## Installation
```
pip install -r requirements.txt
```

## Usage
To run the web application:
```
python app.py
```

## Model
The pre-trained model is available in the `garbage_classifier_cnn.pth` file. The model is a CNN trained on the garbage dataset.

## Dataset
The dataset used for training is not included in this repository due to its size.
